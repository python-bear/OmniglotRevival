# OmniglotRevival
Trial Commit - Devansh
Modernizing Omniglot by updating its underlying server technology.

## Project Goal Statement

Help the creator of the Omniglot website to create new pages and update old pages by updating the backend technology. In addition to this, to create make the information on his websites more machine-readable.

## Plan
These are the things that we need to do for the project:

- Get a full directory tree of the website
- Make a scraper
- Make a database and decide what type to use (probably an SQL)
- Make a GUI so you can add new pages to the database
- Make a generator
- Need a website server

## Setup Python Virtual Environment

1. `python -m venv venv` or `python3 -m venv venv` on macOS.
2. `.\venv\Scripts\activate` use this to actually activate the venv, you'll need to do this every time you open a new terminal. If it worked your command prompt should say 'venv' at the start of it somewhere.
3. `pip install -r requirements.txt` gets all of the project libraries onto your venv.

Use `pip freeze > requirements.txt` to update the requirements if you pip install anything new into the venv.
